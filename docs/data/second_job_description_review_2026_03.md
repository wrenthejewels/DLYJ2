## Project Management Specialists

Reviewed postings:
- Apply Digital - Senior Project Manager
- Avalere Health - Project Manager
- AHEAD - Project Coordinator
- Jobgether - Project Coordinator

Promoted task additions:
- translate project goals into workstreams, cadences, owners, and action tracking
- manage delivery risks, dependencies, and escalations across stakeholders
- maintain project systems, reporting, and budget or resource visibility
- turn delivery status and tradeoffs into decision-ready updates for clients or leaders

Rejected as duplicates:
- generic meeting scheduling
- generic status-note preparation
- generic vendor coordination language already covered by O*NET or manual role-graph tasks
